Comment utiliser cet outil ?

Qu'est-ce qu'Espanso ?

Espanso est un remplaceur de texte (text expander). Il surveille ce que vous tapez et remplace instantanément des mots-clés par du contenu plus long, des dates, ou des formulaires complexes.

À quoi sert cet éditeur ?

Espanso utilise des fichiers .yml pour stocker vos raccourcis. Ce site vous permet de :

  • Créer des raccourcis visuellement sans erreur de syntaxe.
  • Gérer des formulaires avec choix et listes déroulantes.
  • Générer des dates automatiques au bon format.

Où enregistrer vos fichiers ?

Pour qu'Espanso détecte vos raccourcis, enregistrez votre fichier dans le dossier match de votre configuration :

Astuce : Vous pouvez créer autant de fichiers .yml que vous le souhaitez dans ce dossier pour organiser vos raccourcis par thèmes (ex : pro.yml, perso.yml, etc.).

Système Chemin du répertoire /match
Windows %APPDATA%\espanso\match\
macOS ~/Library/Application Support/espanso/match/
Linux ~/.config/espanso/match/

🖥️ Tutoriel : Utiliser des liens symboliques sous Windows

Si vous souhaitez centraliser vos fichiers de raccourcis dans un dossier spécifique (par exemple, pour les synchroniser ou les gérer plus facilement avec l'éditeur), vous pouvez créer un lien symbolique. Cela permettra à Espanso de lire vos fichiers comme s'ils étaient au bon endroit, sans que vous ayez à les déplacer.

Pas de panique, la manipulation ne prend que quelques secondes ! Suivez le guide :

Étape 1 : Ouvrir l'Invite de commande en mode Administrateur

  • Cliquez sur le menu Démarrer de Windows (ou appuyez sur la touche "Windows" de votre clavier).
  • Tapez simplement les trois lettres cmd.
  • Dans les résultats de recherche, faites un clic droit sur Invite de commandes et choisissez Exécuter en tant qu'administrateur.
  • Validez par "Oui" si Windows vous demande une autorisation. Une fenêtre noire s'ouvre.

Étape 2 : Préparer et lancer la commande

Copiez la ligne ci-dessous, mais attention : avant d'appuyer sur Entrée, vous devez remplacer la seconde partie par le chemin du dossier où vous stockez vos raccourcis.

mklink /D "%APPDATA%\espanso\config\match\MonDossier" "C:\Chemin\Vers\Vos\Raccourcis"
💡 Astuce pour ne pas vous tromper de chemin : Allez dans votre explorateur de fichiers, ouvrez le dossier où se trouvent vos raccourcis, cliquez tout en haut dans la barre d'adresse, et copiez le chemin. Remplacez ensuite "C:\Chemin\Vers\Vos\Raccourcis" par ce que vous venez de copier (gardez bien les guillemets).

Étape 3 : Vérifier que cela a fonctionné

Collez la commande modifiée dans la fenêtre noire (un simple clic droit dans la fenêtre suffit souvent à coller le texte) et appuyez sur Entrée.

Si tout est correct, la console affichera ce message de confirmation : Lien symbolique créé pour [...]

Il ne vous reste plus qu'à redémarrer Espanso (en tapant espanso restart dans la console ou via l'icône dans votre barre des tâches) pour qu'il charge vos nouveaux raccourcis.

Note : Pour des raisons de sécurité, les navigateurs ne peuvent pas écrire directement dans vos dossiers système. Enregistrez le fichier sur votre bureau, puis déplacez-le manuellement.
